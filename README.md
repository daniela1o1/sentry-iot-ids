# Sentry

Policy-Driven Intrusion Detection and Automated Response Platform for IoT Systems

## Current MVP

- MQTT attack simulation
- Unauthorized topic publish detection
- Device policy model
- Device trust state foundation
- Docker-based local attack lab

## Planned Features

- Device trust scoring
- Quarantine engine
- Incident store
- Rogue provisioning detection
- Replay attack detection
- Automated response engine

## Stack

- Rust
- MQTT
- PostgreSQL
- Docker
- Axum
- SQLx
